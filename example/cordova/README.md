# AeroGear Services Cordova Example

## Building

Install the dependencies:

```
$ npm install
```

Add platform
```
$ cordova platform add android
$ cordova build android
```

## Running on emulator

Run running platform emulator

```
$ npm run browserify
$ cordova run android
```

## Development in browser

```
## Watch for changes
$ npm run watchify
## In separate terminal
$ cordova run browser
```
